


## Overview

This project is a simple paragraph validation tool created using HTML, CSS, and JavaScript. It allows users to type a paragraph and ensures that the input does not exceed 200 words. The validation is performed on the client side, providing instant feedback to the user.

## Features

- **Text Area:** A user-friendly input area where users can type their paragraph.
- **Word Count Validation:** Automatically counts the words as the user types and prevents the input from exceeding 200 words.
- **Real-Time Feedback:** Provides immediate feedback on the number of words typed, helping users stay within the limit.
- **Responsive Design:** The form is styled with CSS to be visually appealing and works well across different devices and screen sizes.

## Technologies Used

- **HTML:** Used to create the structure of the text input form.
- **CSS:** Used to style the form, making it clean and responsive.
- **JavaScript:** Used for the real-time word count validation, ensuring that the input stays within the specified limit.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/simple-paragraph-validation.git
   ```

2. Navigate to the project directory:
   ```bash
   cd simple-paragraph-validation
   ```

3. Open the `index.html` file in your browser to use the validation tool.

4. Start typing a paragraph in the text area, and the word count will be displayed below. The input will be restricted to 200 words.
